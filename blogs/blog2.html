"A" tag - lets you hyperlink stuff.
"DIV" tag - i use mostly for text.
"IMG" tag - images, duh.
"H1, H2, H3, H4, H5, H6" tags - for headers of different importance.
"P" tag - i barely use this tag but its for paragraphs.
"STRONG, EM" tags - "STRONG" for bolded text, "EM" for itallic.
"TABLE, TBODY, TR, TH, TD" tags - for making tables. "TABLE" and "TBODY" for the start of a table.

"TR" for a new table row. "TH" (must be in a "TR" to work!!) for table headers. "TD" (must be in a "TR" to work!!) for

displaying table elements.
"TITLE" tag - for the title of a webpage.
