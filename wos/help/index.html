All wOS commands!


Any file related commands don't need /wOS/, it's put in automatically!

help - Redirects to this webpage

filemgr(directory or blank) - Opens directory or /wOS/

filenew(file name){file contents} - Creates a new file.

fileedit(file){new file contents} - Edits the contents of a file.

fileview(file) - Views the file in plain text.

fileexport(file) - Exports file as .txt on the user's computer

diskexport - Exports the virtual disk.

diskimport - Imports a virtual disk.

wipedisk - Wipes all data on disk.

terminalclear - Clears all text on the terminal.

echo(text)
