All wOS commands!


Any file related commands don't need /wOS/, it's put in automatically!

cmdhelp - Redirects to this webpage

filemgr(directory or blank) - Opens directory or /wOS/

filenew(file name){file contents} - Creates a new file.

fileedit(file){new file contents} - Edits the contents of a file.

fileview(file) - Views the file in plain text.

fileexport(file) - Exports file as .txt on the user's computer

diskexport - Exports the virtual disk.

diskimport - Imports a virtual disk.

diskwipe - Wipes all data on disk.

terminalclear - Clears all text on the terminal.

echo(text) - Repeats text.









wOS Escape Codes


Yes! wOS has custom characters you can insert!

In order to insert a custom character, you need to hold Escape (ESC)

Then press a number/letter/symbol below!



		  1 - Red Block
		  2 - Orange Block
		  3 - Yellow Block
		  4 - Bright Green Block
		  ` - Teal Block
		  5 - Cyan Block
		  6 - Bright Blue Block
		  7 - Blue Block
		  8 - Purple Block
		  9 - Magenta Block
		  0 - Pink Block
		  "-" - Grey Block
		  = - White Block
